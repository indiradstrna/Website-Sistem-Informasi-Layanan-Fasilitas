## ⚡ Langkah Instalasi (5 menit)

### Step 1 — Salin ke XAMPP

Salin seluruh folder `bioform` ke folder htdocs XAMPP:

```
C:\xampp\htdocs\bioform\
```

### Step 2 — Edit Konfigurasi Database

Buka `config.php` dan sesuaikan:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'root');
define('DB_PASSWORD', '');        // Password MySQL Anda (default XAMPP: kosong)
define('DB_NAME',     'bioform_db'); // Nama database
```

### Step 3 — Pastikan Database Sudah Ada

Database `bioform_db` harus sudah ada. Jika belum:

1. Buka `http://localhost/phpmyadmin`
2. Buat database baru: `bioform_db`
3. Import file `database-schema.sql` dari project Next.js asal

### Step 4 — Aktifkan Apache & MySQL di XAMPP Control Panel

Klik **Start** untuk:
- ✅ Apache
- ✅ MySQL

### Step 5 — Buka di Browser

```
http://localhost/bioform/
```

## 🔑 Akun Default & Role

| Role | Redirect Ke |
|---|---|
| `admin` | `/admin/index.php` |
| `user` | `/user/index.php` |
| `supervisor` | `/supervisor/index.php` |

Akun tersimpan di tabel `users` database. Pastikan sudah ada data user dengan kolom `username`, `password` (plaintext), dan `role`.

**Contoh SQL insert user:**
```sql
INSERT INTO users (full_name, username, password, role)
VALUES ('Administrator', 'admin', 'admin123', 'admin'),
       ('Staff Satu', 'user1', 'user123', 'user'),
       ('Supervisor FMD', 'supervisor', 'svp123', 'supervisor');
```

## ⚠️ Catatan Penting

1. **Password di database masih plaintext** — sesuai sistem existing dari Next.js asli
2. **Tidak ada enkripsi password** — untuk mengaktifkan bcrypt, modifikasi `api/login.php`
3. **Session timeout** — PHP session default 24 menit. Ubah di `php.ini` jika perlu lebih lama
4. **File upload** — jika ada fitur upload foto/dokumen, perlu tambahan konfigurasi
